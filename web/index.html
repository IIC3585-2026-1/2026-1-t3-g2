ℹ️ Algoritmos Disponibles

🔄 Recursión (Fuerza Bruta)

O(2^n) - Muy lento para n grande. Solo para fines educativos.

💾 Memoización (Top-Down)

O(n × W) - Recursión optimizada con caché. Buen balance.

📊 DP Bottom-Up (Recomendado)

O(n × W) - Iterativo. Más eficiente que memoización.

⚡ DP Optimizado

O(n × W), O(W) memoria - Menos RAM, más rápido.
